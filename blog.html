Q. What are the difference between map, forEach, filter and find?

Ans:Map, forEach, filter and find are some javascript array method. The map method takes an array and make a change based on the condition passed through the anonymous function and return a new array. Where as th forEach takes an array and make a change based on the condition passed through the anonymous function but doesn't return a new array. The filter method takes an array and filter the item based on the condition. The find method return the first element from an array that matches the condition passed through the anonymous function
